# Api_Cadastro

## Finalidade

Este projeto foi desenvolvido para gerenciar um sistema simples de cadastro, permitindo registrar e organizar informações de clientes e produtos.

## Funcionalidades

Cadastro de clientes

Cadastro de produtos

Listagem de clientes

Listagem de produtos

Edição e exclusão de registros

## Tecnologias

HTML

CSS

JavaScript

Node.js (se houver backend)

Git e GitHub
